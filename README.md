#Run in dev

1. clon repository git
2. run
```
yarn install  
```
3. Nest Cli should be installed
``` 
npm i -g @nestjs/cli
```

4. db up 
```
docker-compose up -d 
```
4. start
```
yarn start:dev
```

5. restart provisional db
```
localhost:3000/api/v2/seed` 
```
